¡Hola! Soy Lucio Alejandro Argiro, un desarrollador web junior apasionado por construir el lado visible de la web. Mi objetivo es crear experiencias de usuario atractivas y funcionales.

SOBRE MI

Soy un estudiante de Ingeniería en Sistemas que ha descubierto una profunda pasión por el mundo del diseño y el desarrollo web. Mi formación en ingeniería me proporciona una base sólida en la lógica y la resolución de problemas, pero es en la maquetación web donde encontré el lugar perfecto para combinar esa estructura con la creatividad y el diseño centrado en el usuario. Lo que comenzó como curiosidad, rápidamente se convirtió en un interés genuino, impulsado por mis ansias de aprender y de mantenerme al día con las nuevas tecnologías que están moldeando el futuro de la web. Actualmente, me enfoco en perfeccionar mis habilidades en la creación de interfaces intuitivas, responsivas y visualmente atractivas. Mi objetivo es transformar ideas complejas en experiencias digitales que no solo funcionen a la perfección, sino que también sean un placer de usar.

Proyectos.

La cava del chef

Diseño y maquetación de un menú digital para un restaurante de alta cocina, enfocado en una presentación visual elegante y una navegación intuitiva, utilizando únicamente HTML y CSS para demostrar una sólida habilidad en la construcción de interfaces desde cero.

EasyWeb

Ejercicio de maquetación avanzada para replicar el diseño de una agencia web, demostrando un uso intensivo de CSS Flexbox para crear layouts complejos y responsivos de manera precisa, con un fuerte enfoque en la fidelidad visual al diseño original.

Pokedex

Aplicación web interactiva que consume la PokeAPI para mostrar y filtrar datos de Pokémon en tiempo real. Este proyecto demuestra el manejo de JavaScript asíncrono, la manipulación del DOM y la implementación de lógica de búsqueda del lado del cliente.
