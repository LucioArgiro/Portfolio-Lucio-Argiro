La Cava del Chef

Este proyecto consiste en el diseño y la maquetación de un menú digital elegante y funcional para "La Cava del Chef", un restaurante ficticio de alta cocina. Creado exclusivamente con HTML5 y CSS3 puros, el objetivo fue desarrollar una experiencia de usuario intuitiva y visualmente atractiva, que reflejara la identidad sofisticada del restaurante. Este proyecto es una demostración de la capacidad para crear interfaces web estéticas y bien estructuradas desde cero, prestando especial atención a la experiencia del usuario y a la coherencia visual de la marca.

  • HTML
  • CSS
  • JavaScript

EasyWeb Portfolio

Este proyecto es un desafío de maquetación web enfocado en la precisión y el dominio de CSS Flexbox, donde el objetivo fue replicar con la mayor exactitud posible el diseño de una página de inicio de una agencia creativa moderna. Desarrollado utilizando únicamente HTML5 semántico y CSS3 puro, este clon demuestra una sólida comprensión de las técnicas de layout avanzadas sin depender de frameworks externos. Este ejercicio no solo pone de manifiesto la habilidad técnica en HTML y CSS, sino también la atención al detalle y la capacidad para traducir un diseño visual en un código limpio, eficiente y bien estructurado.

  • HTML
  • CSS Flexbox

Pokedex

Este proyecto es una Pokedex interactiva y dinámica, desarrollada con JavaScript puro, que demuestra la manipulación del DOM y el consumo de APIs externas en tiempo real. La aplicación se conecta directamente a la PokeAPI para obtener y mostrar una lista completa de Pokémon, cada uno presentado en una tarjeta individual con su información esencial. El objetivo principal fue crear una experiencia de usuario fluida y responsiva, donde la información se actualiza instantáneamente según la interacción del usuario, en la cual se usaron tecnologías como JavaScript, API Rest y CSS para lograr un diseño atractivo y funcional.

  • JavaScript
  • API Rest
  • CSS
